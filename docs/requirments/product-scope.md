# Product Scope  
Учебный проект — упрощенный аналог Letterboxd.

Документ содержит выделенные **основные** и **второстепенные** фичи на основе анализа оригинального Letterboxd.

---

## 1. Основная цель продукта
Создать платформу, которая позволяет пользователям:
- искать фильмы,
- просматривать информацию о фильмах,
- вести личный трекинг просмотренного,
- оставлять оценки и отзывы,
- формировать списки.

---

## 2. Основные фичи

### 2.1. Аутентификация
- Регистрация пользователя  
- Авторизация  
- Выход из аккаунта  

### 2.2. Каталог фильмов
- Просмотр списка популярных фильмов  
- Поиск фильмов по названию  
- Просмотр детальной страницы фильма:
  - постер  
  - название  
  - год  
  - жанры  
  - описание  
  - средний рейтинг  
  - актёры / команда  

### 2.3. Оценки и отзывы
- Поставить оценку фильму  
- Написать текстовый отзыв  
- Редактировать свой отзыв  
- Удалять собственный отзыв  
- Просматривать отзывы других пользователей  

### 2.4. Watchlist
- Добавить фильм в watchlist  
- Удалить фильм из watchlist  
- Просматривать свой watchlist  

### 2.5. Дневник просмотров
- Добавить фильм как «просмотренный»  
- Указать дату просмотра  
- Просматривать историю просмотров  

### 2.6. Профиль пользователя
- Просмотр собственных данных:
  - оценки  
  - отзывы  
  - список просмотра 
- Редактирование профиля

---

## 3. Второстепенные фичи

### 3.1. Дополнительные данные о фильме
- Трейлеры  
- Похожие фильмы  
- Комментарии к отзывам

### 3.2. Социальные функции
- Просмотр профилей других пользователей  
- Подписки на пользователей  
- Лента активности друзей  
- Упоминания / уведомления  

### 3.3. Создание списков
- Пользовательские коллекции фильмов  
- Описание списка  
- Публичные/личные списки  
- Лайки/комментарии к спискам  

### 3.4. Углубленные настройки
- Смена пароля  
- Изменение аватара  
- Настройки приватности  

---

## 4. Итог
В рамках MVP реализуются функции, необходимые для базового пользовательского опыта: поиск фильмов, просмотр информации, создание оценок и отзывов, ведение дневника и список просмотра, базовый профиль. Все социальные и продвинутые функции переносятся в будущие релизы.
